🌐 Personal Portfolio Website

This is my personal portfolio website built using HTML, CSS, and JavaScript. It showcases my skills, projects, and experience as a web developer in a clean and responsive design.

🔗 Live Website: https://ashhadportfolios.netlify.app/

🚀 Features Fully responsive design (mobile, tablet, desktop) Modern UI with smooth animations Multiple sections (Home, About, Skills, Contact) Interactive navigation menu (hamburger for mobile) Scroll animations and dynamic effects Clean and beginner-friendly code structure 🛠️ Technologies Used HTML5 CSS3 JavaScript (Vanilla JS)

📁 Project Structure Home Page – Introduction and hero section About Page – Personal details and background Skills Page – Technical skills with progress bars Contact Page – Contact form UI

🎯 Purpose

This project was built to improve my frontend development skills and to create a professional online presence showcasing my work and learning journey.
